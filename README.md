# pyxelセーブロードのテストプロジェクト

以下のセーブロードのテスト実装を行なっています。  

* pickleを使用したセーブロード  
* json形式のセーブロード  
* ブラウザ版のローカルストレージを使用したセーブロード  

操作方法は以下の通りです。  

|||
|---|---|
|Z|ダミーのセーブデータのパラメータを設定|
|X|現状のセーブデータのダンプ|
|Q|pickleを使用したセーブ|
|W|pickleを使用したロード|
|A|json形式のセーブ|
|S|json形式のロード|
|E|ブラウザ版のローカルストレージを使用したセーブ|
|R|ブラウザ版のローカルストレージを使用したロード|
